# add_flutter_to_exists_android

android-host: android native project.
flutter-aar: all of flutter aar.
flutter_module: flutter project.

use: 
```bash
cd flutter_module
flutter pub get
cd shell
dart build_module.dart
```

open androi-host in android-studio
run application.
